# Factorial Calculator Using Function

A simple Python program that calculates the factorial of a non-negative integer using a custom function. The program prompts the user to enter a number, computes its factorial by multiplying all positive integers from 1 up to the given number with a `for` loop, and displays the result. It also handles the special case where the input is `0`, returning a factorial value of `1`. This project is ideal for beginners learning about functions, loops, and mathematical calculations in Python.

## Features

* Accepts user input for a non-negative integer.
* Calculates the factorial using a custom function.
* Uses a `for` loop to perform the multiplication.
* Handles the special case of `0! = 1`.
* Displays the factorial result in a clear format.

## How to Run

1. Make sure Python 3 is installed.
2. Run the script:

   ```bash
   python factorial_function.py
   ```
3. Enter a non-negative integer when prompted.
